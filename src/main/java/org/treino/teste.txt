</tbody></table>
<p>Acredita-se que a forma de pirâmides egípcias represente o <a href="/wiki/Benben" title="Benben">monte primordial</a> a partir do qual os egípcios acreditavam que a <a href="/wiki/Terra" title="Terra">Terra</a> foi criada. A forma piramidal representava os <a href="/wiki/Raio_de_luz" class="mw-redirect" title="Raio de luz">raios</a> descendentes do <a href="/wiki/Sol" title="Sol">Sol</a> e a maioria das pirâmides eram revestidas por <a href="/wiki/Pedra_calc%C3%A1ria" class="mw-redirect" title="Pedra calcária">pedra calcária</a> branca, polida e altamente reflexiva, o que dava a essas estruturas uma aparência brilhante quando vistas à distância.<span style="color:gray"><sup>&#91;</sup></span><sup><a href="/wiki/Wikip%C3%A9dia:Livro_de_estilo/Cite_as_fontes" title="Wikipédia:Livro de estilo/Cite as fontes"><span title="Esta afirmação precisa de uma referência para confirmá-la desde junho de 2019." style="color:gray"><i>carece&#160;de fontes</i></span></a><span class="printfooter">?</span><span style="color:gray">&#93;</span></sup>
</p><p>Apesar de geralmente ser aceito que as pirâmides eram monumentos funerários, não há consenso sobre os princípios <a href="/wiki/Teologia" title="Teologia">teológicos</a> particulares que possam ter dado origem a elas. Uma sugestão é que estas estruturas foram concebidas como um tipo de "máquina de ressurreição".<sup id="cite_ref-Não_nomeado-yBMw-2_10-0" class="reference"><a href="#cite_note-Não_nomeado-yBMw-2-10"><span>[</span>10<span>]</span></a></sup>
</p><p>Os egípcios acreditavam que a área escura do céu noturno em torno do qual as estrelas pareciam girar era a porta de entrada física para os céus. Um dos eixos estreitos que se estende a partir da câmara funerária principal por toda a estrutura da <a href="/wiki/Grande_Pir%C3%A2mide" class="mw-redirect" title="Grande Pirâmide">Grande Pirâmide</a> aponta diretamente para o centro desta parte do céu. Isto sugere a pirâmide pode ter sido criada para servir como um meio para lançar magicamente a alma do faraó falecido diretamente para a morada dos deuses.<sup id="cite_ref-Não_nomeado-yBMw-2_10-1" class="reference"><a href="#cite_note-Não_nomeado-yBMw-2-10"><span>[</span>10<span>]</span></a></sup>
</p><p>Todas as pirâmides egípcias foram construídas na margem ocidental do <a href="/wiki/Rio_Nilo" title="Rio Nilo">rio Nilo</a>, local onde ocorre o <a href="/wiki/P%C3%B4r-do-sol" class="mw-redirect" title="Pôr-do-sol">pôr-do-sol</a>, o que era associado ao reino dos mortos na mitologia egípcia.<sup id="cite_ref-11" class="reference"><a href="#cite_note-11"><span>[</span>11<span>]</span></a></sup>
</p>
